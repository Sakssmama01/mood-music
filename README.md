# 🎧 Mood Music Recommender

An interactive web app that recommends YouTube music playlists based on your current mood — built using HTML, CSS, and JavaScript.

---

## 🧠 Features

- 🎵 Mood-based music suggestions (Happy, Sad, Chill, Energetic)
- 🖼️ Song cards with thumbnails and titles
- 🌗 Dark mode toggle
- 💻 Fully responsive design (mobile-friendly)
- 🎬 Direct YouTube links to play songs
